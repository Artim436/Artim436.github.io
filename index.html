Equation normale

For \(A \in \mathbb{R}^{m \times n}\), \(b \in \mathbb{R}^m\) and \(x \in \mathbb{R}^n\), the normal equation is given by

\[ A^\top A x = A^\top b \]

Existence

There always exists a solution to the normal equation.

Proof

First, we prove that \(\text{Ker}(A^\top A) = \text{Ker}(A)\).

It is clear that \(\text{Ker}(A) \subset \text{Ker}(A^\top A)\).

Then for \(x \in \text{Ker}(A^\top A)\), we have

\[ A^\top A x = 0 \Rightarrow x^\top A^\top A x = 0 \Rightarrow \| Ax\|_2^2 = 0 \Rightarrow Ax = 0 \]

Thanks to the rank theorem, we have that

\[ \text{dim(Ker)}(A) = \text{dim(Ker)}(A^\top A) \]

By recalling that \(\text{rank}(A^\top ) = \text{rank}(A)\) (justified with the \(J_r\) decomposition), we conclude that

\[ \text{dim(Ker)}(A^\top) = \text{dim(Ker)}(A) = \text{dim(Ker)}(A^\top A) \]

Secondly, we prove that \(\text{Im}(A^\top A) = \text{Im}(A^\top)\).

Clearly, \( \text{Im}(A^\top A) \subset \text{Im}(A^\top) \).

Since \(\text{dim(Ker)}(A^\top) = \text{dim(Ker)}(A^\top A)\), the rank theorem tells us

\[ \text{rank}(A^\top A) = \text{rank}(A^\top) < +\infty \]

By the property of finite-dimensional subspaces, we conclude that \(\text{Im}(A^\top A) = \text{Im}(A^\top)\).

Finally, since

\[ A^\top b \in \text{Im}(A^\top) = \text{Im}(A^\top A) \]

there exists \(x\) such that \( A^\top A x = A^\top b \).

Remarks

  • \(x\) is a solution to the normal equation if and only if it minimizes the least square error \( \| Ax - b \|_2^2 \).
  • For a full rank matrix \(A\), there exists a unique solution: \[ x = (A^\top A)^{-1} A^\top b \]
  • \( x = (A^\top A)^+ A^\top b \) belongs to the set of solutions and minimizes the norm of \(x\).
