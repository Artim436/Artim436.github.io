Equation normale : existence of a solution

For \(A \in \mathbb{R}^{m \times n}\), \(b \in \mathbb{R}^m\) and \(x \in \mathbb{R}^n\), the normal equation is given by:

\[ A^\top A x = A^\top b \]

Existence

There always exists a solution to the normal equation.

Proof

First, we prove that \(\text{Ker}(A^\top A) = \text{Ker}(A^\top)\).

It is clear that \(\text{Ker}(A^\top) = \text{Ker}(A) \subset \text{Ker}(A^\top A)\). Then for \(x \in \text{Ker}(A^\top A)\), we have:

\[ A^\top A x = 0 \Rightarrow x^\top A^\top A x = 0 \Rightarrow ||Ax||^2_2 = 0 \Rightarrow Ax = 0 \]

Secondly, we want to prove that \(\text{Im}(A^\top A) = \text{Im}(A^\top)\).

It is clear that \( \text{Im}(A^\top A) \subset \text{Im}(A^\top)\).

Then, because \(\text{Ker}(A^\top A) = \text{Ker}(A^\top)\), the Rank theorem tells us that \(\text{dim(Im)}(A^\top A) = \text{dim(Im)}(A^\top) < +\infty \).

By the property of the sub vectorial space of finite dimension, we can conclude that \(\text{Im}(A^\top A) = \text{Im}(A^\top)\).

Finally, we have that:

\[ A^\top b \in \text{Im}(A^\top) = \text{Im}(A^\top A) \]

So, there exists \(x\) such that \(A^\top A x = A^\top b\).

Remarks

  • This solution is unique if and only if \(A\) is full rank. In that case: \[ x = (A^\top A)^{-1} A^\top b \]
  • \(x\) is a solution to the normal equation if and only if \(x\) minimizes the least square error \(||Ax - b||_2^2\).
